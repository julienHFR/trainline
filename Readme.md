## Setup

$ yarn install
or
$ npm i

### Build

$ yarn build
or
$ npm run build

### Start

$ yarn start
or
$ npm run start

### Test

$ yarn test
or
$ npm run test

### Distant reachable server

$ yarn server
or
$ npm run server
